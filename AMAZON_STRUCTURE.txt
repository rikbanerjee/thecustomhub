================================================================================
                              WORKFLOW
================================================================================

1. UPDATE PRODUCT DATA
   └─> Edit src/data/products.json
       └─> Add marketplace.amazon = "active"
       └─> Add amazonData object
       └─> Add specifications
       └─> Add dimensions

2. TEST DATA
   └─> Run: node src/amazon/test.js
       └─> Validates fields
       └─> Shows mappings
       └─> Identifies issues

3. GENERATE FEED
   └─> Run: npm run amazon:generate
       └─> Reads products.json
       └─> Validates data
       └─> Generates Excel
       └─> Generates TSV

4. UPLOAD TO AMAZON
   └─> Go to Amazon Seller Central
       └─> Catalog > Add Products via Upload
       └─> Upload: output/amazon/amazon_upload_ready.tsv

================================================================================
                        REQUIRED FIELDS
================================================================================

✓ SKU                       - Product identifier
✓ Product Type              - drinking_cup
✓ Item Name                 - Product title
✓ Brand Name                - Your brand
✓ Main Image URL            - Primary image
✓ Price                     - Selling price
✓ Quantity                  - Inventory count
✓ Condition                 - New/Used
✓ Fulfillment Channel       - DEFAULT or AMAZON_NA
